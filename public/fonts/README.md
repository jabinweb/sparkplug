# Supermolot Neue Trl Font Files

This directory should contain the Supermolot Neue Trl font files as specified in the Sparkplug Brand Guidelines.

## Required Font Files

Place the following font files in this directory:

- `SupermolotNeue-Light.woff2` (weight: 300)
- `SupermolotNeue-Regular.woff2` (weight: 400)
- `SupermolotNeue-Medium.woff2` (weight: 500)
- `SupermolotNeue-SemiBold.woff2` (weight: 600)
- `SupermolotNeue-Bold.woff2` (weight: 700)

## Font Format

The configuration uses `.woff2` format for optimal web performance. If you have `.ttf` or `.otf` files, you can convert them to `.woff2` using online tools or font conversion software.

## License

Ensure you have the proper licensing for Supermolot Neue Trl before deploying to production.

## Fallback

The website is configured with fallback fonts in case Supermolot Neue is not available:
- Supermolot Neue (system-installed version)
- -apple-system
- BlinkMacSystemFont
- Segoe UI
- system-ui
- sans-serif
